# QuadroCount
A combinatorial Game I had designed as a part of my course IE 619
It is a two player game played on a m*n board game
The Ruleset is as follows:
• Starting position: 2 chips of colours red and blue placed anywhere on the
board
• The game is characterized by the size of the rectangle formed by the two
pieces
• The two pieces can be considered as the ends of a rectangle of size 1*n or
the diagonal ends
• In any move, a player can move one piece, similar to how we move a king
in chess to increase the size of the rectangle formed by them
• The size of the rectangle formed by them must increase by at least 1 unit
in each move
• The game ends when a player cannot increase the size of the rectangle
anymore
• The game is not about the size of the rectangle being formed, but more
about the ability of a player to increase the size of rectangle
